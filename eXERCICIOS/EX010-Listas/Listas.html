Trabalhando Com Listas

Listas ordenadas

Existe um tag chamada <ol> que faz uma lista ordenada (ordened list). Dentro da tag, cada ítem da lista é colocada entre a tag <li>, que significa list iten. Além disso, caso necessite personalizar a lista, pode-se usar o padrão "type" dentro de ol e escolher se a lista vai ser ordenada com números, letras, etc. Pode-se, ainda, escolher a partir de qual número/letra a lista vai iniciar utilizando a opção "start" depois de type.

  1. Acordar
  2. Tomar café
  3. Banhar
  4. Ler

Lista não ordenadas

São as que não precisam de ordenação, de sequenciação. Usa-se a tag <ut>, que significa "unorded list". Pode-se também configurar a personalização usando o padrão "type" dentro de "ul" e escolhendo qual tipo de símbolo que vai ser usado na lista. Os tipos são "circle, disk e square".

  • Comprar arroz
  • Comprar feijão
  • Tomate
  • Banana
  • Frango

Listas Mistas

São meio que junções entre listas. É colocar uma lista dentro de outra lista. Por exemplo, é possível fazer uma lista e colocar outra lista (subtópicos) dentro de cada item. Basta adicionar uma nova "ol" dentro de "li"

  1. Vingadores novos favoritos
    1. Pantera Negra
    2. Homem Formiga
    3. Wanda
  2. Vingadores antigos favoritos
    1. Homem de Ferro
    2. Thor
    3. Hulck

    Além disso, pode-se misturar uma "ol" com uma "ul".

    1. Iphone
      • Camera ótima
        1. Fotos muito boas
        2. Melhor qualidade de vídeos
      • Gravar tela sem precisar de outro app
        1. Facilidade e agilidade com gravação de tela
    2. Android
      • Apps crackeados
        1. Não precisar gastar tanto com apps pagos
        2. Ter apps sem anúncios
      • Variedade de jogos grátis
        1. Não precisar pagar por um jogo qualquer que queira jogar
        2. Encontrar jogos pagos de graça
        3. Não precisa comprar armazenamento
        4. Pode-se até expandir o armazenamento de graça em alguns celulares

Listas de Definição

Ela funciona como se fosse um dicionário. Ela é literalmente uma lista de termos e definições. Para isso, usa-se a tag <dl> para a criação da lista. Dentro de "dl", coloca-se os termo através da tag <dt> e, dentro de "dt", coloca-se as definições através da tag <dd>. Ela dá valor semântico para o arquivo e auxília o site a ser encontrado quando algúem estiver pesquisando por uma definição. Ela pode ser configurada em CSS para melhorar a aparência.
