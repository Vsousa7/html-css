Estilos Externos

Os estilos externos são os bem mais recomendados para quando você precisa padronizar diversos códigos (h1,h2,h3,p,body,etc) em um bom número de paginas. Ele é essencial para isso, pois permite que toda a sua parte de estilo fique externa a sua parte HTML, fazendo com que todos os estilos de todas as páginas do seu site sejam configuradas apenas com uma folha de estilo.

A folha de estilo

Folha de estilo é o nome do arquivo que é usado em CSS para configurar seus estilos.É nela que todas as configurações da tag style(que antes eram feitas em head) irão ficar. Como o estilo fica em body no HTML, na folha de estilo é preciso colocar o body como seletor.

Para adicionar a stylesheet (folha de estilo), basta adicionar a tag <link: css>. O arquivo da folha de estilo precisa estar na mesma pasta do index para faciliar mais na hora de adicioná-la na tag.

Na stylesheet, a primeira coisa a se fazer é definir a regra @charset "UTF-8". O CSS funciona com regras, representadas pelos @. Elas ficam sempre no início do arquivo.
