Principais formatações

Negrito/destaque

Nessa frase, temos um termo em negrito usando a taga <b>, não semântica.

Nessa frase, há um termo em destaque usando a tag <strong>, com semântica, dando um sentido de termo forte na frase.

Itálico/destaque

Nessa frase teremo um termo em itálico usando a tag <i>, não semântica.

Nesta frase temos um termo em ênfase usando a tag <em>, com semântica.

Nesta frase, usaremos a tag <small> para deixar as letras pequenas.

Nesta frase veremos texto deletado usando a tag <del>

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usando a tag <ins>

Pode-se utilizar também as tags <U> e <span style="text-decoration: underline"> para deixar um texto sublinado, mas sem semântica

Pode-se acionar textos sobreescritos usando a tag <sup> e textos subescreitos usando a tag <sub>
