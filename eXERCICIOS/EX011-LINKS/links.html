Usando links externos

O que são os links externos

Links externos são links que vão apontar para outros sites, que vão te levar para algum site no momento em que você clica no texto com link.

Exemplo: você pode acessar meu repositório público no Github. A tag <a> vem de "âncora". Em "hypertext reference" coloca-se a url do site.

Vou colocar agora um link para o canal no YouTube do Tolezuando.

Adicionando links externos sem o navegador sobrepor seu site

Na forma anterior de adição de link externo, ao clicar no link, o navegador sobrepõe o site externo ao nosso site. Isso faz com que as pessoas provavelmente nunca mais voltem ao nosso site. Por isso, é necessário usar alguns parâmetros na tag "a" para fazer com que o navegador abra o link externo em outra aba.

Para isso, faz-se assim: você vai clicar no link do canal do Tolezuando e o navegador vai abrir em outra aba. A função "target" é um tipo de alvo; a função "_blank" é uma janela em branco; e a função "rel" significa "relativo à". Logo, toda essa função significa que o link externo vai ter um alvo em _blank, e o link vai ser aberto em outra janela. Mas o "rel" é algo mais novo. Nem sempre é necessário.

Usando links internos

Eles fazem ligação, relação com uma página que está dentro do nosso servidor e diz respeito a nossa prórpia página. Com isso, para ter um link interno é necessário ter um outra página. Crie ela no mesmo lugar onde cria o index.

Esta é a primeira página do site. A seguir, estará o link para a segunda página do site. Para uma página que faz parte do meu servidor, do meu site por completo, não precisa de link. Precisa apenas que minhas outras páginas estejam salvas na mesma pasta onde minha página princial (index) está.

O "rel" em links externos usa uma função diferente. Nele, temos que adicionar a função "next" (que indica um avanço para uma próxima página. Nesse caso, indica a segunda página) e "prev" (que indica uma página anterior). Essa função serve para escolher qual será a próxima página e, assim, o Google não vai definir isso por você.
