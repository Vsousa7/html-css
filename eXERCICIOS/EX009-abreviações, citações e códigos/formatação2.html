Outras formatações

Código Fonte

São os códigos de programação. Para uma melhor leitura, suas letras precisam ocupar o mesmo espaço, ou seja, ter fontes monoespaçadas. Para isso, usa-se a tag <code>

Para códigos muito grandes (como os do Phyton), é necessário colocar a tag CODE dentro da tag <pre> para deixar o código com as devidas identizações. Ex: 


>>> complex(0, 0)
Tudo o que estiver dentro de PRE vai ser considerado com suas quebras de linha, idetinzações, etc. Ela não é a mais correta, mas é uma tag de pré-formatação.

Citações

Como diria o grande filósofo Piton: tudo o que você faz na vida é pra comer alguém. Essa citação, o acréscimos das aspas, foi fetia utilizando a tag <q>. Para citações assim pequenas, essa tag é bem recomendada. Ela é utilizada para dar sentido semâncitco no HTML. Ou seja, ela é usada para indicar ao navegado que o que está entre ela é uma citação. Ela não é decorativa.

Cirações completas

Há também citações que são mais longas. Para isso, usa-se a tag <blockquote>. Ex: Segundo Mantoan (2002, pg. 2):

“Entre a fundação desse Instituto e os dias de hoje, a história da educação especial no Brasil foi se estruturando, seguindo quase sempre modelos que primam pelo assistencialismo, pela visão segregativa e por uma segmentação das deficiências, fato que contribui ainda mais para que a formação escolar e a vida social das crianças e jovens com deficiência aconteçam em um mundo à parte”.

Se essa citação tiver um link, pode-se adicioná-lo usando o padrão "cite="xxxxx" dento de blockquote. Isso é essencial para uma citação mais completa e para que os navegadores consigam identificar melhor seu site e as citações.

Abreviações (ou sigla)

Para fazer com que os navegadores identifiquem, semanticamente, que determinada palavra é uma silga/abreviação, é preciso adicionar a tag <abbr> entre a palavra abreviada. Ex: HTML. A função title na tag serve para colocar o significado da sigla/abreviação.

Texto invertido

A tag <bdo> serve para inverter um texto. Ex: se você leu até aqui é pq é g4y.
